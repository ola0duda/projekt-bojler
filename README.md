# Analizator Ścieżki Krytycznej (Metoda CPM)
Projekt aplikacji webowej służącej do planowania harmonogramów projektów przy użyciu metody **Critical Path Method (CPM)**. Aplikacja pozwala na wyznaczanie czasów najwcześniejszych i najpóźniejszych dla zadań, identyfikację ścieżki krytycznej oraz wizualizację wyników w formie grafu i wykresu Gantta.

Propozycja: React (Frontend) + Node.js (Backend)
